Associative Arrays

Turns out, Javascript can't acatully use associative arrays. Which means Named Indexes by the way, it has to use Numbered Indexes. However JavaScript Objects do use Named Indexes, but we'll get to them later.

This one works just fine:

		
var person = [];
person[0] = "John";
person[1] = "Doe";
person[2] = 46;
var x = person.length;         // person.length will return 3
var y = person[0];             // person[0] will return "John"

This one just breaks everything:

		
var person = [];
person[0] = "John";
person[1] = "Doe";
person[2] = 46;
var x = person.length;         // person.length will return 3
var y = person[0];             // person[0] will return "John"
